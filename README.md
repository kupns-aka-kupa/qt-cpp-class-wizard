### Usage:

#### [Manual](https://doc.qt.io/qtcreator/creator-project-wizards.html).

* 1:

```bash
    git clone https://github.com/kupns-aka-kupa/qt-cpp-class-wizard.git

    ./install.sh
```

* 2: **Restart** qtcreator.
* 3 Enjoy.
